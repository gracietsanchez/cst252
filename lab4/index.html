Challenge

This lab was meant to show us how to develop the process of pseudocoding a simple task and then translating the same process to a computer task. .

Problems

We ran into the small problem of figuring out what kind of computer task we wanted to pseudocode. It was difficult selecting a game, task, or automation without knowing how such tasks should be broken down when programming. However, after some research and contemplation, we selected a simple computer task that we felt comfortable dividing into high-level tasks.

Results

We successfully outlined and pseudocoded an everyday task as well as a computer task. The pseudocode for the everyday task describes the process behind making strawberry brownies, and the pseudocode for the computer task describes a program asking for the user's name in order to create a personalized greeting. We turned both outlines into comments in our lab.js folder and included them in our lab4/index.html as well. Finally, we published our lab4 pages to the server.
